# autojs_web

## 介绍

> 使用 html css js 作为 autojs 的界面
>
> 当然也可以用 react vue  之类的 前端框架

## 使用

> 在 vscode 中 按 Ctrl + Shift + P 
>
> 选择 `Auto.js Pro 运行项目(Run Project)`  即可
>
> 代码中有例子 ( vue3 + vant ). 可以直接进行测试
